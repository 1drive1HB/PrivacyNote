# (https://1drive1hb.github.io/PrivacyNote/)

- 🔒 Client-side encryption (AES-256)
- ⏱️ Self-destructing notes
- 📱 Mobile-friendly UI
- 🌓 Dark mode support
- 📋 One-click copy & share
